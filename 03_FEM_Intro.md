## FEM 和 PBD / XPBD 最大的不同

PBD / XPBD 的思路更像：

- 先预测位置
- 再把位置投影回约束集合

FEM 的思路更像：

- 先描述物体如何形变
- 写出形变能量
- 对能量求导得到力
- 再更新系统

所以 FEM 更接近连续介质力学和现代弹性仿真。

## 最小入口：2D 三角形

建议从二维三角形单元开始，不要一上来就碰三维四面体。

### 参考构型

初始位置：

- `X1, X2, X3`

### 当前构型

当前变形后的位置：

- `x1, x2, x3`

### 形变梯度

先构造：

```math
D_m = [X_2 - X_1,\; X_3 - X_1]
```

```math
D_s = [x_2 - x_1,\; x_3 - x_1]
```

然后：

```math
F = D_s D_m^{-1}
```

这里的 `F` 就是在描述：

**参考单元是如何被映射到当前单元的。**

## FEM 的核心链条

```text
参考构型 -> 当前构型 -> deformation gradient F -> strain energy psi(F) -> force
```

## 你现在先不用硬啃的东西

先不要一上来追：

- 张量分析
- 完整连续介质力学教材
- 大量材料模型细节

先抓最核心的对象：

- `F`
- `psi(F)`
- `force = -dE/dx`
